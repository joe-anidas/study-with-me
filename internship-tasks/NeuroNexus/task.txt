TASK 1 - LANDING PAGE

Create a Google Form-like web form using only HTML and CSS (no JavaScript).
The form must collect the following details:
• Full Name (Text Input)
• Contact Number (Phone Input)
• Address (Multiline Textarea)
• Company Name (Text Input)
• Email Address (Email Input)
• Department (Dropdown/Select Menu with options like HR, Sales, Development, Marketing, Others)
• Gender (Radio Buttons: Male, Female, Prefer not to say)
• Submit Button

Bonus (Optional for extra points):
• Add a simple hover effect on the submit button.
• Make the background color of the form different from the page background.
• Add a minimalistic box-shadow to the form.
